# 🎯 Pro Portfolio - محفظة استثمارية احترافية

مرحباً بك في تطبيق **Pro Portfolio**! تطبيق ويب احترافي لإدارة محفظتك الاستثمارية بسهولة وأمان.

## 📋 المحتويات

✅ **صفحة التسجيل والدخول** (index.html)
✅ **لوحة التحكم الرئيسية** (dashboard.html)
✅ **رسوم بيانية متقدمة**
✅ **جدول إدارة الأسهم**
✅ **مؤشرات الأداء الرئيسية (KPIs)**

---

## 🚀 كيفية الاستخدام

### **الخطوة 1: الملفات المطلوبة**

ستحتاج إلى ملفين فقط:
1. `index.html` - صفحة التسجيل
2. `dashboard.html` - اللوحة الرئيسية

### **الخطوة 2: الاختبار المحلي**

1. احفظ الملفات في نفس المجلد
2. افتح ملف `index.html` في المتصفح
3. **إنشاء حساب جديد:**
   - أدخل اسمك
   - بريدك الإلكتروني
   - كلمة مرور (6 أحرف على الأقل)

4. **تسجيل الدخول:**
   - استخدم بيانات حسابك

5. **من اللوحة الرئيسية:**
   - اضغط "إضافة سهم جديد"
   - أدخل بيانات السهم
   - شاهد الرسوم البيانية والمؤشرات تتحدث تلقائياً

---

## 📊 المميزات الحالية

### **المؤشرات (KPIs)**
- 💰 **إجمالي المشتريات** - مجموع الأموال المستثمرة
- 📈 **إجمالي الأرباح** - الأرباح/الخسائر بالكامل
- 📊 **القيمة الحالية** - قيمة محفظتك الآن

### **الرسوم البيانية**
1. **أداء المحفظة** - رسم خطي يوضح الأرباح/الخسائر
2. **توزيع الاستثمار** - رسم دائري يوضح توزيع الأموال
3. **أداء الأسهم** - رسم أعمدة يوضح النسبة المئوية لكل سهم

### **جدول الأسهم**
- اسم السهم وسعر الشراء والكمية
- السعر الحالي والقيمة الحالية
- الأرباح/الخسائر والنسبة المئوية
- الفئة والملاحظات
- أزرار تعديل وحذف

---

## 🔐 نظام التخزين الحالي

حالياً، يتم تخزين البيانات محلياً (LocalStorage):
- **المستخدمون** → `localStorage.users`
- **الأسهم** → `localStorage.stocks`
- **المستخدم الحالي** → `localStorage.currentUser`

⚠️ **ملاحظة:** البيانات محفوظة على جهازك فقط، لا تُفقد عند الإغلاق

---

## 🔄 الخطوات التالية (مع Firebase)

عندما تكون جاهزاً للخطوة التالية:

### **المرحلة 2: إضافة Firebase**

1. **إنشاء حساب Firebase:**
   - اذهب إلى https://firebase.google.com
   - ملخص وسهل (مجاني 100%)

2. **ربط التطبيق:**
   - استبدال LocalStorage بـ Firebase Database
   - إضافة Firebase Authentication
   - تشفير البيانات وتأمينها

3. **المميزات الجديدة:**
   - ✅ البيانات محفوظة على الانترنت
   - ✅ الوصول من أي جهاز
   - ✅ مزامنة فورية
   - ✅ أمان عالي

### **المرحلة 3: النشر على الانترنت**

- **GitHub Pages** - مجاني وسهل
- أو **Firebase Hosting** - من نفس الحساب

## 📞 الدعم والمساعدة

### **المشاكل الشائعة:**

**المشكلة:** البيانات تُحذف عند حذف cache المتصفح
**الحل:** هذا طبيعي مع LocalStorage - سنستخدم Firebase لاحقاً

**المشكلة:** الرسوم البيانية لا تظهر
**الحل:** تأكد من أن Chart.js محمّل بشكل صحيح

**المشكلة:** التطبيق لا يعمل على الموبايل
**الحل:** استخدم متصفح حديث (Chrome, Firefox, Safari)
